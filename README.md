# Personal Portfolio 🌐

This is my personal portfolio website built with **Vite**, **Vanilla JavaScript**, and **Tailwind CSS**. It showcases my projects, skills, and contact information — all in a responsive, modern layout.

## 🚀 Features

- ⚡ Fast development with [Vite](https://vitejs.dev/)
- 🎨 Styled with [Tailwind CSS](https://tailwindcss.com/)
- 📱 Responsive design
- 🔗 Live project links + GitHub repo buttons
- 📸 Project screenshots
- 📬 Contact section
